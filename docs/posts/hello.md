# 用 VitePress 搭建个人博客

这是博客的第一篇示例文章。你可以把它当成写作模板：每篇文章都放在 `docs/posts/` 目录下，并在开头写好 frontmatter。

## 推荐结构

一篇文章通常包含：

- 标题、发布日期、摘要和标签
- 问题背景
- 解决过程
- 最终结论
- 后续可以继续优化的地方

## 写作建议

不用一次写得很完整。先把问题和结论记录下来，再逐步补充过程、代码片段和参考链接。个人博客最重要的是持续积累。
